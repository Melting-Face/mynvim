## Install

```shell
gem install tmuxinator
```

<br>

## Project

### Create new project

```shell
tmuxinator new [project]
```

<br>

### Start new project

```shell
tmuxinator start [project]
```

<br>

### windows 

  <b>root</b>

    - tmux pane workspace path

  <b>layout</b>

    - even-horizontal

    - even-vertical

    - main-horizontal

    - main-vertical

    - tiled

  <b>pane</b>

    - execution programs
