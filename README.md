## Install

### nvim + tmux
```shell
sh nvim.sh all
# or
sh nvim.sh
```

### Plugins to be installed
[packer](https://github.com/wbthomason/packer.nvim)

### install sdkman for kotlin
```shell
brew install gradle
curl -s "https://get.sdkman.io" | bash
source "$HOME/.sdkman/bin/sdkman-init.sh"
sdk install kotlin
```
